# Documentacao do Sistema GMF

Esta pasta centraliza a documentacao empresarial do sistema de controle de hidrometros, estoque, almoxarifado e rastreabilidade operacional da GMF.

## Documentos principais

- [Documentacao tecnica do codigo](DOCUMENTACAO_TECNICA.md): arquitetura, estrutura do projeto, banco de dados, seguranca, auditoria, rotas, servicos, testes, deploy e manutencao.
- [Manual de uso do sistema](MANUAL_USO_SISTEMA.md): guia operacional por perfil, com explicacao das telas, fluxos, boas praticas e orientacao para registrar erros pela area de Feedback.

## Regra operacional de suporte

Sempre que o usuario identificar erro, duvida operacional, comportamento estranho, informacao incorreta ou dificuldade de uso, deve registrar pela area **Feedback** do sistema.

O Feedback deve ser marcado como urgente quando:

- impedir a operacao do dia;
- afetar estoque, caixas, hidrometros, pecas, instaladores ou solicitacoes;
- gerar divergencia operacional;
- envolver acesso indevido, erro de permissao ou falha de seguranca;
- impedir relatorio, backup, auditoria ou rastreabilidade.

## Publico-alvo

- **Administracao e TI:** usar a documentacao tecnica para sustentacao, deploy, auditoria, correcao controlada e evolucao segura.
- **Usuarios do sistema:** usar o manual operacional para saber o que fazer em cada tela e quando acionar o Feedback.
- **Gestao:** usar ambos como base para treinamento, implantacao, homologacao e padronizacao do processo.
